# investrategy

A modern, interactive 3D financial planning website built with React, Three.js, and glassmorphism design — inspired by Apple aesthetics and content from financial life planning best practices.

## Features

- **3D Hero Scene** — Floating glass orbs, particle field, and iridescent rings powered by React Three Fiber
- **Glassmorphism UI** — Liquid glass panels with backdrop blur, subtle gradients, and Apple-inspired typography
- **Interactive Sections** — Life stage tabs, scroll-driven process timeline, and live financial calculators
- **Merged Content** — Financial life planning, wealth management, SIP/investment guidance, educational resources, and workflow process

## Tech Stack

- React 19 + TypeScript
- Vite
- Three.js / React Three Fiber / Drei
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
